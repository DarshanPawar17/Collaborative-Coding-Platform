# Collaborative Code Editor

A real-time collaborative code editor built with React, Monaco Editor, Yjs, and Socket.IO.

## Getting Started

1. Navigate to the Frontend and Backend directories and install dependencies using `npm install`.
2. Start the Backend server using `npm run start` or `node server.js` (runs on port 3000).
3. Start the Frontend development server using `npm run dev`.
4. (Optional) Create a `.env` file in the Frontend to set `VITE_BACKEND_URL` if your backend is hosted elsewhere.
